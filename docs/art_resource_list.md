# 아트 리소스 목록

## 1. 목적

이 문서는 대장장이 고양이 MVP와 출시 버전에 필요한 아트/사운드 리소스를 정리한다.
초기 구현에서는 임시 리소스로 대체할 수 있지만, 코드 구조는 실제 리소스를 교체하기 쉽게 구성한다.

## 2. 전체 아트 스타일

- 2D 픽셀아트
- 귀엽고 캐주얼한 분위기
- 옛날 플래시 게임 감성
- 작은 화면에서도 명확한 실루엣
- 과도한 디테일보다 반복 플레이 가독성 우선

## 3. 해상도 기준

권장 기준:
- 캐릭터 원본: 128x128 또는 256x256 픽셀 감성
- 무기 원본: 64x64 또는 128x128
- UI 아이콘: 32x32 또는 64x64
- 배경: 세로형 모바일 기준 1080x1920에 맞게 확장 가능

실제 파일은 고해상도 PNG로 넣더라도 픽셀아트 느낌을 유지한다.

## 9. 앱 아이콘

앱 아이콘 콘셉트:
- 대장장이 고양이 얼굴 + 작은 망치
- 배경은 따뜻한 대장간 주황/갈색 계열
- 고양이와 망치 실루엣이 작게 봐도 보여야 함
- 텍스트 없음

필요 크기:
- 1024x1024 원본
- Android adaptive icon 대응 가능 구조
- iOS 아이콘 대응 가능 구조

## 11. 초기 임시 리소스 처리

실제 리소스가 없을 경우:
- 고양이: 이모지 또는 간단한 픽셀 박스
- 무기: 색이 다른 사각형/간단한 CustomPaint
- 이펙트: Flutter 기본 애니메이션
- 사운드: 생략 가능

단, 파일명/ID 기준은 위 문서를 따라 나중에 교체하기 쉽게 만든다.

## 12. 우선 제작 순서

1. 앱 아이콘
2. 고양이 기본/망치질/성공/실패
3. 무기 0~20 단계 대표 이미지
4. 메인 화면 UI 패널/버튼
5. 성공/실패 이펙트
6. 대장간 배경
7. 사운드
