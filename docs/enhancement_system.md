# 강화 시스템 기획서

## 1. 기본 방향

대장장이 고양이의 핵심은 `강화해서 팔기`이다. 복잡한 전투나 던전보다 강화 버튼을 누르는 순간의 긴장감과 결과 보상이 중심이다.

## 2. 강화 단계

### 전체 목표
- 장기 최종 강화: +99
- 에피소드 1 범위: +1 ~ +20

### 에피소드 1 구간 구분

| 구간 | 이름 | 특징 |
|---|---|---|
| +1 ~ +5 | 입문 강화 | 높은 성공률, 실패 부담 거의 없음 |
| +6 ~ +10 | 일반 강화 | 성공률 하락, 재료 관리 시작 |
| +11 ~ +15 | 마력 주입 강화 | 다른 강화 방식 도입 |
| +16 ~ +20 | 불안정 강화 | 보호권/특수 재료 가치 증가 |

## 3. 강화 방식

### +1 ~ +10: 일반 강화
필요 요소:
- 골드
- 철 조각 또는 기본 재료

실패 패널티:
- +1 ~ +5: 골드와 재료만 소모
- +6 ~ +10: 낮은 확률로 강화 수치 -1 또는 재료 추가 손실

의도:
초반은 가볍게 성공 경험을 주고, +6부터 살짝 긴장감을 준다.

### +11 ~ +20: 마력 주입 강화
+10 이후에는 단순 골드 강화가 아니라 `마력 주입`이라는 다른 규칙을 도입한다.

필요 요소 후보:
- 골드
- 마력석
- 같은 단계 또는 이전 단계 검
- 보호 부적

추천 방식:
- +11 ~ +15: 골드 + 마력석 필요
- +16 ~ +20: 골드 + 마력석 + 이전 단계 무기 조각 필요

실패 패널티:
- +11 ~ +15: 실패 시 수치 유지 또는 -1
- +16 ~ +20: 실패 시 -1, 낮은 확률로 파손

파손은 캐주얼 게임에서는 스트레스가 크므로 기본 무기 완전 삭제보다는 `부서진 검`으로 변환하는 방식을 추천한다.

## 4. 재료 구조

### 기본 재료
- 철 조각: +1 ~ +10 강화에 사용
- 고급 철 조각: +6 ~ +10에서 추가 사용 가능
- 마력석: +11 이후 사용
- 별가루: 보호권 제작 또는 고급 강화에 사용
- 부서진 검 조각: 분해/실패 보상으로 획득

### 재료 획득 경로
1. 상점에서 골드로 구입
2. 무기 분해로 획득
3. 실패 보상으로 일부 획득
4. 광고 보상 또는 일일 보상으로 획득

## 5. 무기 분해

무기를 팔지 않고 분해하면 골드는 적게 얻지만 재료를 얻는다.

### 분해 규칙 예시
- +0 ~ +5: 철 조각 획득
- +6 ~ +10: 철 조각 + 고급 철 조각 획득
- +11 ~ +15: 마력석 일부 획득
- +16 ~ +20: 마력석 + 별가루 낮은 확률 획득

분해는 플레이어가 `돈이 필요한지, 재료가 필요한지` 선택하게 만드는 장치다.

## 6. 강화 결과

### 성공
- 강화 수치 +1
- 무기 가치 증가
- 일정 구간마다 외형 변화
- 고양이 기쁨 애니메이션

### 실패
- 재료와 골드 소모
- 구간에 따라 수치 유지, 하락, 파손 중 하나
- 고양이 당황/울상 애니메이션
- 너무 긴 실패 연출 금지

### 대성공 후보
낮은 확률로 +2 강화 또는 추가 가치 상승.
초기 MVP에서는 제외 가능하지만, 강화 게임의 재미를 위해 데이터 필드만 미리 준비하면 좋다.

## 7. 보호 아이템

### 깨짐 방지권
- 파손 결과를 막는다.
- +16 이후부터 가치가 커진다.
- 광고 보상/상점/이벤트로 제공 가능.

### 하락 방지권
- 실패 시 강화 단계 하락을 막는다.
- +11 이후부터 사용 가능.

### 시작 강화권
- 새 무기를 +3, +5, +10 등에서 시작하게 한다.
- 플래시 게임 감성의 편의 아이템.
- 너무 강하면 게임이 짧아지므로 에피소드 1에서는 +3 시작권, +5 시작권 정도만 추천.

## 9. 추천 설계 결론

에피소드 1은 복잡한 제작 게임이 아니라 강화 게임이어야 한다. 그러므로 재료 종류는 적게 유지하고, +10 이후에만 다른 강화 규칙을 보여준다.

추천 MVP 재료:
- 골드
- 철 조각
- 마력석
- 검 조각
- 보호권

추천 MVP 실패 규칙:
- +1 ~ +5: 소모만
- +6 ~ +10: 낮은 확률 하락
- +11 ~ +15: 하락 가능
- +16 ~ +20: 하락 + 낮은 파손 가능
