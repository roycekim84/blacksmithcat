# 연출 및 피드백 기획서

## 1. 목적

강화 게임은 버튼 하나의 감각이 매우 중요하다.
이 문서는 강화 시도, 성공, 실패, 판매, 분해, 상점 구매 등 주요 행동의 피드백을 정의한다.

## 2. 기본 원칙

- 연출은 짧고 반복을 방해하지 않아야 한다.
- 성공은 기분 좋게, 실패는 아쉽지만 다시 누르고 싶게 만든다.
- 중요한 단계(+5, +10, +15, +20)는 일반 성공보다 강한 연출을 준다.
- 정보는 숨기지 않고 즉시 보여준다.

## 3. 강화 버튼 피드백

### 버튼 누름
순서:
1. 강화 버튼이 눌리는 애니메이션
2. 고양이가 망치를 들어올림
3. 모루에 망치 타격
4. 짧은 플래시/불꽃
5. 성공 또는 실패 결과 표시

권장 시간:
- 전체 0.6 ~ 1.0초
- 반복 강화를 위해 1초를 넘기지 않는 것이 좋다.

## 4. 성공 연출

### 일반 성공
사용 구간:
- 대부분의 강화 성공

연출:
- 작은 불꽃 이펙트
- 무기 살짝 점프
- 강화 수치가 +1로 튀어오르는 텍스트
- 고양이 기쁨 표정

문구 예시:
- `강화 성공!`
- `검이 더 단단해졌습니다!`
- `대장장이 고양이의 눈이 반짝입니다.`

### 구간 돌파 성공
사용 구간:
- +5
- +10
- +15

연출:
- 일반 성공보다 큰 불꽃
- 화면 살짝 흔들림
- 무기 외형 변화 강조
- 짧은 축하 팝업

문구 예시:
- `+5 달성! 검의 형태가 바뀌었습니다.`
- `+10 달성! 마력 주입 강화가 열렸습니다.`
- `+15 달성! 검이 신비한 마검으로 변했습니다.`

### 에피소드 클리어 성공
사용 구간:
- +20

연출:
- 별빛 폭발
- 화면 중앙에 전설검 표시
- 고양이 특별 기쁨 애니메이션
- 에피소드 클리어 팝업

문구 예시:
- `+20 첫 번째 전설검 완성!`
- `작은 대장간에 첫 전설이 탄생했습니다.`

## 5. 실패 연출

### 일반 실패
연출:
- 작은 연기
- 무기가 흔들림
- 고양이 당황 표정

문구 예시:
- `강화 실패...`
- `재료가 사라졌습니다.`
- `아직 불꽃이 부족했던 것 같습니다.`

### 하락 실패
연출:
- 균열 이펙트
- 강화 수치가 -1로 내려가는 표시
- 고양이 놀람 표정

문구 예시:
- `강화 실패! 검의 힘이 약해졌습니다.`
- `강화 단계가 1 내려갔습니다.`

### 파손 실패
연출:
- 큰 균열
- 짧은 연기
- 무기가 부서진 검으로 바뀜
- 고양이 울상 표정

문구 예시:
- `검이 부서졌습니다!`
- `하지만 쓸만한 검 조각을 건졌습니다.`

파손 연출은 너무 과하게 공포스럽거나 짜증나게 만들지 않는다.

## 6. 보호권 사용 피드백

### 하락 방지권 발동
문구:
- `하락 방지권이 검의 힘을 붙잡았습니다!`

연출:
- 방패 아이콘이 잠깐 나타남
- 강화 수치는 유지

### 깨짐 방지권 발동
문구:
- `깨짐 방지권이 파손을 막았습니다!`

연출:
- 보호막 이펙트
- 파손 이펙트가 보호막에 막히는 느낌

## 7. 판매 피드백

### 일반 판매
연출:
- 골드 아이콘이 상단 골드 영역으로 이동
- 판매가 숫자 증가 애니메이션

문구:
- `무기를 판매했습니다.`
- `골드 +{amount}`

### 고강화 판매
+10 이상 판매 시 확인 팝업:
- `강화 수치가 높은 무기입니다. 정말 판매할까요?`

## 8. 분해 피드백

연출:
- 무기가 조각으로 흩어짐
- 획득 재료 아이콘 표시

문구:
- `무기를 분해했습니다.`
- `철 조각 +{amount}`
- `마력석 +{amount}`
- `검 조각 +{amount}`

+10 이상 분해 시 확인 팝업:
- `강화 수치가 높은 무기입니다. 정말 분해할까요?`

## 9. 재료 부족 피드백

강화 버튼을 눌렀지만 재료가 부족한 경우:

문구:
- `철 조각이 부족합니다.`
- `마력석이 부족합니다.`
- `골드가 부족합니다.`

UI:
- 부족한 재료를 빨간색 또는 흔들림으로 강조
- `상점으로 이동` 버튼 제공

## 10. +10 전환 연출

+10 달성 후 다음 강화부터 강화 방식이 바뀐다.

연출:
1. +10 성공 팝업
2. 대장간 화로가 푸른빛으로 변화
3. `마력 주입 강화가 열렸습니다!` 문구 표시
4. 강화 버튼 문구가 `강화하기`에서 `마력 주입`으로 변경

## 11. 반복 플레이 편의

### 빠른 재강화
결과 연출 후 바로 다시 강화할 수 있어야 한다.
팝업이 있을 경우 `다시 강화` 버튼을 크게 둔다.

### 자동 결과 닫기
일반 성공/실패는 짧은 토스트로 처리하고, 구간 돌파만 팝업으로 처리해도 좋다.

추천:
- 일반 결과: 토스트/상단 배너
- +5/+10/+15/+20: 팝업
- 파손: 팝업

## 12. 진동 피드백

가능하면 모바일 진동을 사용한다.

- 버튼 클릭: 아주 약한 진동
- 성공: 짧은 진동
- 실패: 짧고 둔한 진동
- 파손: 조금 더 강한 진동

설정에서 ON/OFF 가능해야 한다.

## 13. 사운드 피드백

- 강화 시도: 망치 소리
- 성공: 밝은 효과음
- 실패: 둔탁한 효과음
- 파손: 깨지는 소리, 단 너무 날카롭지 않게
- 판매: 골드 소리
- 구매: 가벼운 클릭/획득 소리

## 14. 우선 구현 순서

1. 강화 버튼 눌림 애니메이션
2. 성공/실패 텍스트 피드백
3. 골드/재료 숫자 변화 애니메이션
4. +5/+10/+15/+20 팝업
5. 파손 팝업
6. 보호권 발동 표시
7. 사운드/진동
