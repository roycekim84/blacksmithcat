# 에피소드 1 기획서 - 작은 대장간의 첫 검

## 1. 목표

에피소드 1은 전체 게임의 맛보기이자 출시 가능한 첫 콘텐츠다.

범위:
- 강화 단계: +1 ~ +20
- 무기 종류: 검 1종 중심
- 핵심 시스템: 강화, 판매, 분해, 상점, 보호권
- 플레이 목표: +20 검 제작

## 2. 에피소드 콘셉트

작은 마을의 대장간에서 신비한 능력을 가진 고양이가 첫 번째 마법 검을 완성하는 이야기.

처음에는 평범한 낡은 검을 다루지만, 강화가 진행될수록 검에 불꽃, 마력, 별빛이 깃든다.

## 3. 진행 구간

### +0 ~ +5: 낡은 검 다듬기
- 기본 강화 튜토리얼
- 성공률 높음
- 실패해도 큰 손실 없음
- 무기 외형은 낡은 검에서 반짝이는 철검으로 변화

### +6 ~ +10: 장인의 검 만들기
- 성공률이 조금씩 낮아진다.
- 고급 철 조각 도입 가능.
- 실패 시 낮은 확률로 -1 하락.
- +10 달성 시 첫 번째 큰 연출.

### +11 ~ +15: 마력 주입
- 강화 방식이 바뀐다.
- 마력석 필요.
- 대장간 화로 색이 푸른빛/보랏빛으로 변한다.
- 무기에 오라가 생긴다.

### +16 ~ +20: 불안정한 마검
- 검 조각 또는 별가루 필요.
- 하락/파손 위험 도입.
- 보호권 사용 가치 증가.
- +20 달성 시 에피소드 클리어 연출.

## 4. 무기 외형 단계

| 강화 구간 | 무기 이름 예시 | 외형 |
|---|---|---|
| +0 | 낡은 검 | 녹슨 회색 검 |
| +1 ~ +4 | 손질된 검 | 밝아진 철색 |
| +5 | 단단한 철검 | 날이 선명해짐 |
| +6 ~ +9 | 장인의 검 | 손잡이 장식 추가 |
| +10 | 불꽃 철검 | 작은 불꽃 이펙트 |
| +11 ~ +14 | 마력 깃든 검 | 푸른 오라 |
| +15 | 신비한 마검 | 보라색 빛 |
| +16 ~ +19 | 불안정한 별검 | 별가루 이펙트 |
| +20 | 첫 번째 전설검 | 강한 빛과 전용 연출 |

## 5. 튜토리얼 흐름

1. 고양이가 기본 검을 보여준다.
2. 강화 버튼을 누르게 한다.
3. 성공 결과와 판매가 상승을 보여준다.
4. 몇 번 강화 후 판매를 안내한다.
5. 판매 골드로 재료를 사게 한다.
6. +6 이후 실패 가능성을 안내한다.
7. +10 달성 후 마력 주입 강화를 안내한다.
8. +16 이후 보호권을 안내한다.

튜토리얼은 강제 팝업을 최소화하고, 말풍선 1~2줄로 짧게 진행한다.

## 6. 에피소드 1 클리어 조건

- +20 무기 제작 성공

클리어 보상:
- 특별 칭호: 첫 전설을 만든 고양이
- 대장간 배경 장식 1개
- 다음 에피소드 예고 화면
- +5 시작권 1개

## 7. 에피소드 1 MVP 콘텐츠 목록

### 시스템
- 강화
- 판매
- 분해
- 상점
- 보호권 사용
- 저장/불러오기

### 화면
- 메인 강화 화면
- 상점 화면
- 인벤토리/무기 관리 화면
- 도감 또는 기록 화면
- 설정 화면

### 데이터
- +1 ~ +20 강화 테이블
- 무기 외형 단계 데이터
- 판매가 테이블
- 재료 드랍/분해 테이블
- 상점 상품 테이블

## 8. 에피소드 1에서 제외해도 되는 것

- 여러 무기 타입
- 손님 의뢰 시스템
- 대장간 꾸미기 심화
- 복잡한 장비 옵션
- PvP/랭킹
- 실시간 전투
- 긴 스토리 컷신

## 9. 첫 출시 기준

첫 출시 버전은 +20까지 만드는 재미가 확실하면 충분하다.

핵심 체크:
- 강화 버튼을 누르는 맛이 있는가?
- 실패해도 다시 하고 싶은가?
- 팔기 타이밍이 고민되는가?
- +10 이후 변화가 느껴지는가?
- +20 달성 연출이 기분 좋은가?
