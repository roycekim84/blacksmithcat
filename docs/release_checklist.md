# 출시 체크리스트

## 1. 목적

이 문서는 대장장이 고양이를 실제 모바일 앱으로 출시하기 전 확인해야 할 항목을 정리한다.

## 2. MVP 기능 완료 체크

### 핵심 루프
- [ ] 새 게임 시작 가능
- [ ] 현재 무기 표시
- [ ] +1 ~ +20 강화 가능
- [ ] 강화 비용 차감
- [ ] 재료 차감
- [ ] 성공/실패 처리
- [ ] 구간별 실패 패널티 적용
- [ ] +10 이후 마력 주입 강화 전환
- [ ] +20 달성 처리

### 판매/분해
- [ ] 현재 무기 판매 가능
- [ ] 판매가 정상 지급
- [ ] 판매 후 새 무기 지급
- [ ] 현재 무기 분해 가능
- [ ] 분해 보상 정상 지급
- [ ] +10 이상 판매 확인 팝업
- [ ] +10 이상 분해 확인 팝업

### 상점
- [ ] 철 조각 구매
- [ ] 마력석 구매
- [ ] 검 조각 구매
- [ ] 하락 방지권 구매
- [ ] 깨짐 방지권 구매
- [ ] +3 시작권 구매
- [ ] +5 시작권 구매
- [ ] 골드 부족 처리

### 보호권
- [ ] 하락 방지권 사용 가능
- [ ] 하락 발생 시에만 소모
- [ ] 깨짐 방지권 사용 가능
- [ ] 파손 발생 시에만 소모
- [ ] 보호권 부족 처리

## 3. 저장/불러오기 체크

- [ ] 앱 종료 후 골드 유지
- [ ] 앱 종료 후 재료 유지
- [ ] 앱 종료 후 현재 무기 유지
- [ ] 앱 종료 후 최고 강화 기록 유지
- [ ] 앱 종료 후 기록 통계 유지
- [ ] 앱 업데이트 후 기존 저장 데이터 유지 가능성 검토
- [ ] 데이터 초기화 기능 동작

## 4. UI/UX 체크

- [ ] 세로 모드 고정
- [ ] 작은 화면에서도 강화 버튼이 잘 보임
- [ ] 상단 재화 표시가 잘림 없이 보임
- [ ] 성공률/비용/판매가가 명확히 보임
- [ ] 재료 부족 시 부족한 항목이 명확히 표시됨
- [ ] 일반 성공/실패 연출이 너무 길지 않음
- [ ] +5/+10/+15/+20 연출이 구분됨
- [ ] 팝업 닫기/다시 강화 동선이 빠름
- [ ] 한 손 조작 가능

## 5. 밸런스 체크

- [ ] +5까지 너무 오래 걸리지 않음
- [ ] +10까지 초반 목표로 적절함
- [ ] +11 이후 재료 전환이 이해됨
- [ ] +16 이후 보호권 가치가 느껴짐
- [ ] +20 달성이 너무 쉽거나 너무 어렵지 않음
- [ ] 판매와 분해 선택이 의미 있음
- [ ] 상점 가격이 과도하지 않음
- [ ] 재료 부족으로 플레이가 막히는 시간이 너무 길지 않음

## 6. 아트/사운드 체크

- [ ] 앱 아이콘 준비
- [ ] 스플래시 이미지 준비
- [ ] 고양이 기본 이미지 준비
- [ ] 고양이 망치질 이미지 또는 애니메이션 준비
- [ ] 성공/실패 표정 준비
- [ ] 무기 단계별 이미지 준비
- [ ] 대장간 배경 준비
- [ ] UI 버튼/패널 준비
- [ ] 강화 성공/실패 효과음 준비
- [ ] 설정에서 사운드 ON/OFF 가능
- [ ] 설정에서 진동 ON/OFF 가능

## 7. 광고/BM 체크

초기 출시에서 광고를 붙일 경우:

- [ ] 보상형 광고 SDK 연동
- [ ] 광고 로드 실패 처리
- [ ] 광고 보상 중복 지급 방지
- [ ] 일일 광고 제한 저장
- [ ] 광고 제거 상품 여부 결정
- [ ] 개인정보 처리방침에 광고 SDK 명시

초기 출시에서 광고를 붙이지 않을 경우:

- [ ] 광고 버튼 숨김 또는 준비중 처리
- [ ] 광고 관련 코드가 게임 진행을 막지 않음

## 10. 테스트 시나리오

### 기본 시나리오
1. 새 게임 시작
2. +5까지 강화
3. 무기 판매
4. 재료 구매
5. +10까지 강화
6. 마력 주입 전환 확인
7. +15까지 강화
8. 분해 후 재료 획득 확인
9. 보호권 구매
10. +20 도전

### 실패 시나리오
- 골드 부족 상태에서 강화 시도
- 재료 부족 상태에서 강화 시도
- 보호권 부족 상태에서 보호권 선택
- +10 이상 판매 취소
- +10 이상 분해 취소
- 파손 발생
- 저장 후 앱 재시작

## 11. 출시 전 최종 질문

출시 직전 아래 질문에 모두 답할 수 있어야 한다.

- 이 게임의 재미가 10초 안에 전달되는가?
- 강화 버튼을 계속 누르고 싶은가?
- 실패해도 다시 해볼 마음이 드는가?
- +10 이후 변화가 분명한가?
- +20 달성이 기분 좋은가?
- 광고가 게임 흐름을 망치지 않는가?
- 첫 버전 범위가 너무 크지 않은가?
