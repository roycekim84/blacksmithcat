# UI 플로우 기획서

## 1. 기본 방향

대장장이 고양이는 세로형 모바일 게임이다.
한 손으로 플레이할 수 있어야 하며, 핵심 버튼은 엄지로 누르기 쉬운 하단 또는 중앙 하단에 배치한다.

## 2. 핵심 화면 수

출시 전 MVP 기준 5개 화면을 추천한다.

1. 메인 강화 화면
2. 상점 화면
3. 인벤토리/무기 관리 화면
4. 도감/기록 화면
5. 설정 화면

## 3. 메인 강화 화면

### 목적
게임의 80% 이상이 이루어지는 핵심 화면.

### 구성
상단:
- 골드
- 주요 재료 보유량
- 현재 에피소드/최고 강화 기록

중앙:
- 대장장이 고양이
- 모루 위 현재 무기
- 강화 단계 표시
- 판매 예상가 표시
- 성공 확률 표시

하단:
- 강화 버튼
- 판매 버튼
- 분해 버튼
- 보호권 토글/선택
- 상점/인벤토리 이동 버튼

### 연출
- 강화 버튼 클릭 시 고양이가 망치를 든다.
- 성공 시 별/불꽃 이펙트.
- 실패 시 연기/삐걱 이펙트.
- +5, +10, +15, +20에서는 특별 연출.

## 4. 상점 화면

### 목적
재료와 편의 아이템 구매.

### 구성
- 재료 탭
- 보호권 탭
- 시작권 탭
- 광고 보상 탭 또는 무료 보상 영역

### 상품 예시
- 철 조각 10개
- 마력석 5개
- 검 조각 5개
- 하락 방지권
- 깨짐 방지권
- +3 시작권
- +5 시작권

## 5. 인벤토리/무기 관리 화면

### 목적
보유 중인 무기와 재료 확인.

### 구성
- 현재 무기
- 보관 무기 목록
- 재료 목록
- 분해 가능한 무기 선택

MVP에서는 무기 보관을 복잡하게 만들지 않고, 현재 무기 중심으로 시작해도 된다.

## 6. 도감/기록 화면

### 목적
플레이어의 수집/성취 욕구를 자극한다.

### 구성
- 달성한 최고 강화
- 제작한 무기 외형 목록
- +5, +10, +15, +20 기록
- 총 판매 골드
- 총 강화 시도 수
- 총 성공/실패 수

MVP에서는 `기록 화면`으로 단순하게 시작하고, 나중에 도감으로 확장한다.

## 7. 설정 화면

### 구성
- 사운드 ON/OFF
- 진동 ON/OFF
- 알림 설정
- 데이터 초기화
- 크레딧
- 개인정보/약관 링크

## 9. UX 원칙

### 강화 버튼은 가장 커야 한다
강화 게임의 메인 액션이므로 가장 눈에 띄어야 한다.

### 결과는 빠르게 보여준다
성공/실패 연출은 0.5~1초 안에 끝난다.

### 확률과 비용은 숨기지 않는다
성공률, 비용, 실패 패널티는 명확히 표시한다.

### 판매/분해는 실수 방지 필요
고강화 무기를 판매/분해할 때는 확인 팝업을 띄운다.
예: +10 이상 판매/분해 시 확인.

## 10. 추가 화면 후보

- 이벤트 화면
- 에피소드 선택 화면
- 고양이 의상 화면
