# 밸런스 테이블 기획서

## 1. 목적

이 문서는 에피소드 1(+1 ~ +20) MVP 구현을 위한 초기 밸런스 기준이다.

수치는 확정값이 아니라 첫 구현용 기준값이다. 실제 플레이 테스트 후 조정한다.
단, Codex는 구현 시 아래 테이블을 우선 적용하고, 코드에 하드코딩하지 말고 데이터 테이블로 분리한다.

## 2. 밸런스 방향

### 플레이 감각 목표
- +5: 매우 쉽게 도달
- +10: 초반 목표 지점
- +15: 재료 관리가 필요한 중간 목표
- +20: 에피소드 1의 첫 장기 목표

### 실패 스트레스 기준
- +1 ~ +5: 거의 스트레스 없음
- +6 ~ +10: 약간 긴장
- +11 ~ +15: 하락 위험으로 긴장
- +16 ~ +20: 보호권 사용을 고민하는 구간

## 3. 재화 단위

### 기본 재화
- gold: 골드
- iron: 철 조각
- magic_stone: 마력석
- blade_piece: 검 조각
- protect_down: 하락 방지권
- protect_break: 깨짐 방지권

## 5. 실패 처리 규칙

실패 시 기본적으로 사용한 골드와 재료는 소모된다.

### 실패 타입
- consume_only: 비용과 재료만 소모
- downgrade_chance: 일정 확률로 강화 단계 -1
- break_chance: 일정 확률로 파손

### 파손 처리
에피소드 1에서는 파손 시 무기를 완전히 삭제하지 않는다.
대신 다음 중 하나로 처리한다.

추천 MVP 처리:
- 현재 무기는 +0 낡은 검으로 돌아간다.
- 검 조각을 일부 지급한다.
- 결과 메시지: `검이 부서졌지만 쓸만한 조각을 건졌습니다.`

이 방식은 강화 게임의 긴장감은 유지하면서 모바일 캐주얼 스트레스를 줄인다.

## 6. 보호권 규칙

### 하락 방지권
사용 가능 구간:
- +6 이상

효과:
- 실패 시 발생하는 -1 하락을 1회 막는다.
- 사용 여부는 강화 버튼 근처에서 선택한다.
- 실패했지만 하락이 발생하지 않은 경우에도 소모할지 여부는 구현 정책을 정해야 한다.

추천 정책:
- 하락 결과가 실제로 발생했을 때만 소모한다.

### 깨짐 방지권
사용 가능 구간:
- +16 이상

효과:
- 파손 결과를 1회 막는다.
- 파손이 실제로 발생했을 때만 소모한다.
- 하락까지 같이 막지는 않는다.

## 7. 시작권 규칙

### +3 시작권
효과:
- 새 무기를 +3 상태로 시작한다.
- 기본 무기 판매/분해/파손 후 새 무기 생성 시 사용 가능.

### +5 시작권
효과:
- 새 무기를 +5 상태로 시작한다.
- 초반 반복 피로를 줄이기 위한 아이템.

### 제한
- 시작권은 +10 이상으로 제공하지 않는다.
- 에피소드 1에서는 +3, +5만 사용한다.

## 12. 테스트 체크리스트

- +1 강화 비용과 재료가 정상 차감되는가?
- 재료 부족 시 강화가 막히는가?
- 성공률이 테이블 기준으로 적용되는가?
- 실패 시 구간별 패널티가 적용되는가?
- 하락 방지권이 하락 발생 시에만 소모되는가?
- 깨짐 방지권이 파손 발생 시에만 소모되는가?
- 판매가가 현재 강화 단계 기준으로 지급되는가?
- 분해 보상이 현재 강화 단계 기준으로 지급되는가?
- +20 달성 시 에피소드 클리어 처리가 되는가?
