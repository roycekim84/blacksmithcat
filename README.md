# 대장장이 고양이

세로형 모바일 캐주얼 강화 게임.

## 한 줄 컨셉
신비한 능력을 가진 대장장이 고양이가 무기를 강화하고, 강화된 무기를 팔아 돈을 벌며 더 높은 강화 단계와 새로운 제작법을 열어가는 픽셀아트 모바일 게임.

## 핵심 플레이
1. 무기 준비
2. 재료와 골드로 강화 시도
3. 성공하면 강화 수치 상승 및 외형 변화
4. 실패하면 하락/파손/재료 손실 등 리스크 발생
5. 원하는 시점에 무기를 판매해 골드 획득
6. 상점/분해/제작으로 다음 강화 준비

## 출시 방향
- 플랫폼: 모바일 앱
- 화면 방향: 세로형
- 스타일: 픽셀아트, 캐주얼, 플래시게임 감성
- 장르: 무기 강화 / 수집 / 판매
- 에피소드 1 범위: +1 ~ +20 강화
- 최종 장기 목표: +99 강화
